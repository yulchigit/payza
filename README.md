# PayZa_asosiy

PayZa is a fintech web app for payments, wallet dashboard, merchant analytics, and crypto/card conversion flows.

## Tech stack
- React 18
- Vite 5
- Redux Toolkit
- React Router v6
- Tailwind CSS

## Run locally
```bash
npm install
npm run start
```

Frontend API base URL (example):
```bash
VITE_API_BASE_URL=http://localhost:5000/api
```

## Build
```bash
npm run build
```
